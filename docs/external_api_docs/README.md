# External API Documentation

This directory contains documentation for external libraries used in the PhenoTag project.

## Libraries

- [Streamlit](./streamlit/): Interactive web app framework
- [DuckDB](./duckdb/): In-process SQL OLAP database management system
- [Polars](./polars/): Fast DataFrame library
- [Marimo](./marimo/): Reactive notebook library
- [OpenCV](./opencv/): Computer vision library

Each subdirectory contains markdown documents organized by topic, making it easier to find and reference documentation.
